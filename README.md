# Introduction:

# Problem Statement:

Employ Computer Vision AI that analyzes gameplay frames and automatically controls paddle movement in a Breakout game using real-time visual input by tracking hand movements.

# Solution:

Use various CV models trained separately to find the best hand movement tracking (mAP50-95 > 0.9) from a live webcam stream. Use the trained model to atuomatically control paddel movements in order to play the BREAKOUT game.

MediaPipe, generic YOLO models, and YOLO Pose
